## 🚀 What Is This Extension?

The **Hetzner Cloud Toolkit** is an unofficial, community-maintained VS Code extension that allows you to:

- ✅ Create and manage servers without leaving your editor
- ✅ Manage networks, SSH keys, and Infrastructure as Code
- ✅ Auto-inject Tailscale for secure VPN access
- ✅ Support multiple Hetzner Cloud projects
- ✅ All credentials stored securely in VS Code's SecretStorage

## 🎯 Key Features at a Glance

| Feature | Description |
|---------|------------|
| **7-Step Server Wizard** | Guided flow to create servers with all options |
| **Multi-Project Support** | Manage multiple Hetzner Cloud projects from one extension |
| **Network Management** | Create and manage private networks with subnets |
| **SSH Key Management** | Manage SSH keys with built-in key generation guide |
| **Tailscale Auto-Inject** | Automatically add Tailscale to cloud-init on server creation |
| **Secure Token Storage** | All API tokens encrypted in VS Code's SecretStorage |
| **Real-Time Status** | Live server status indicators (running, off, initializing) |
